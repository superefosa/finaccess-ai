# Testing Strategy

## Automated tests
- Authentication registration and login
- Protected route access using bearer tokens
- Financial profile creation
- Loan application creation and AI scoring
- Admin review flow
- Recommendation threshold mapping

## Manual test checklist
1. Register a normal user.
2. Log in and save a financial profile.
3. Submit a loan application.
4. Confirm score, recommendation, and explanation appear.
5. Log in as admin.
6. Approve or reject the application.
7. Confirm the user dashboard reflects the final decision.
